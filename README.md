## dotfiles

---

Daniel Landers dotfiles, currently local OSX only

Run the following command to bootstrap a new machine

```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/danlannz/dotfiles/master/bootstrap.sh)"
```
